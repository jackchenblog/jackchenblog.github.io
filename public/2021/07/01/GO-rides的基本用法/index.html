安装Redis

启动Redis

连接Redis

例如：port=8081
–raw 可以输出中文

Redis基本命令

存

取

删

DEL key

序列化

DUMP key

检查给定 key 是否存在

EXISTS key
1: 存在
0: 不存在

为给定 key 设置过期时间

以秒计

EXPIRE key seconds

毫秒

PEXPIRE key milliseconds

时间戳-毫秒

PEXPIRE key milliseconds

移除 key 的过期时间，key 将持久保持

PERSIST key

显示key距离过期剩余时间-毫秒

PTTL key

显示key距离过期剩余时间-秒

TTL key

随机返回一个key

RANDOMKEY

修改Key

RENAME key newkey
替换newkey的值 key置空

RENAMENX key newkey
当newkey存在时，重命名失败

返回 key 所储存的值的类型

TYPE key
redis 中没有int值，会使用string存入

Redis String 基本命令

存取

返回 key 中字符串值的子字符

GETRANGE key start end
类似于字符串截取, 从哪个下标开始截取到哪里，包含end

将给定 key 的值设为 value ，并返回 key 的旧值

对 key 所储存的字符串值，获取指定偏移量上的位(bit)

当偏移量 OFFSET 比字符串值的长度大，或者 key 不存在时，返回 0

获取所有(一个或多个)给定 key 的值

将值 value 关联到 key ，并将 key 的过期时间设为 seconds

以秒为单位
SETEX key seconds value
以毫秒为单位
PSETEX key milliseconds value

只有在 key 不存在时设置 key 的值

SETNX key value

覆盖偏移量之后的字符

SETRANGE key offset value

返回 key 所储存的字符串值的长度

STRLEN key

同时设置一个或多个 key-value 对

覆盖已有key 的value
MSET key value [key value …]
当且仅当所有给定 key 都不存在
MSETNX key value [key value …]

在对应的string值后面追加字符

APPEND key value
