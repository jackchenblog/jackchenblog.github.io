交叉编译(跨平台编译)

  • 平台: windows/mac/linux三大平台
  • 简单说就是在windows. 上编译成可以在ma/linux.上直接运行的可执行文件，反之亦然。
  • 默认执行go build编译的可执行文件都是当前操作系统上课运行的文件。
  • 如果需要编译出在其他平台，上运行的可执行文件，需要指定目标操作系统的平台和处理器架构

直接在对用的终端下执行 SET XXXX 命令即可实现,go build 之后会生成对应平台的文件
